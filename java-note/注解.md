# 注解-Annotation

> - 注解可以 标注在 ： 方法、属性、类       ...
> 
> - 注解可 不改变原有代码，“嵌在代码之上”，做补充
> 
> - 注释只给程序员看； 注解给【编译器】 or 【java虚拟机】看...
> 
> - 所有的注解 是继承：java.lang.annotation.Annotation     //是接口
